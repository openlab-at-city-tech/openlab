Easily embed a Google Doc into your blog posts

== Description ==

Okay, so you've written an awesome doc, created a killer presentation, made some nifty calculations in a spreadsheet or made a fillable form on Google Docs.

Now, you want to embed that Google Doc into a blog post.  What to do?

Use this plugin!

Simply enter in the Google Docs shortcode into your blog post and presto, the doc should embed into your blog post!

To view a tutorial on how to use the shortcode, [read this](http://github.com/cuny-academic-commons/google-docs-shortcode#google-docs-shortcode).

== Installation ==

1. Download, install and activate this plugin.
1. To view a tutorial on how to use the shortcode, [read this](http://github.com/cuny-academic-commons/google-docs-shortcode#google-docs-shortcode).
